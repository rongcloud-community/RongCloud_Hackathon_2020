安装说明:

软件版安装

软件版已经集成Apache2.2、Tomcat5.5和MySQL5.0,
点击安装程序setup.exe安装完成后会自动打开淘客服务器管理和淘客数据初始化页面（http://127.0.0.1/install），
在进行数据初始化之前，请检查淘客服务器管理中的所有服务（apache,tomcat和mysql）处于启动状态,
同时请确定系统启动tomcat的用户对webapp/root目录有写权限。


站点版安装

打开淘客数据初始化页面（http://你的域名/install），
在进行数据初始化之前，请检查你的站点支持jsp，数据库已经创建和访问数据库的用户已经创建并分配包含创建/删除表的权限，,
同时请确定用户对站点目录有写权限。
