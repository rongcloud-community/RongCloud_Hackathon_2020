# 项目介绍  
## 队名  
下一个队  

## 项目名  
SChat  

## 项目简介  
本项目展现了学习社区的场景，使用到融云即时通讯的sdk实现了一个聊天室，实现学习社区使用过程中用户和用户之间的交流。  

## 使用说明  
运行程序后访问localhost：8080/index,可进入主页页面，点击讨论区，即可进入聊天与其他用户进行实时交流（在详情界面需要在url后加上用用户的id信息“?userId=xxx”）。
