# 生存游戏中添加和朋友的视频交流功能
## 适用场景
#### 如果朋友之间不再一起，可以通过网络同时玩游戏和与好友聊天
#### 另外一方修改客户端接口，即可实现在游戏中看游戏直播教程，可同时在游戏内验证
#### 为了方便使用，也是直接修改了UE4插件后，添加了音视频功能
#### 游戏为末日生存游戏，类似于七日杀，僵尸会不断围观主角
## 开发
 项目基于Unreal Engine 和 Agora video c++ sdk 开发，包含Agore UE4插件和 Web 客户端与 UE4演示程序
